readme.txt of the newuoa toolbox

The first version of the toolbox was written by Matthieu GUILBERT (matthieu.guilbert@inrialpes.fr).
The conversion to Scilab 5 has been performed by Yann COLLETTE (ycollet@freesurf.fr)

execute:

exec builder.sce;

to build the toolbox.
Once the toolbox is built, each time you want to load the toolbox, execute:

exec loader.sce;
