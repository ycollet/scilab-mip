SciSYMPHONY Module

This toolbox allows you to use the (mixed integer) linear programming tools under scilab.

To use this toolbox under scilab: go into the scisymphony module installation directory and do a:

exec loader.sce;

Once the toolbox has been loaded, close and reopen the help browser to access to the documentation of the functions.
You have also access to a demo via the ?->Demonstrations of Scilab button from the GUI.
